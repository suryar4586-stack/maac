# StackOS — Frontend Only

Windows 11-inspired browser OS built purely in React. No backend required.
All data (files, settings, notes) persists in **localStorage**.

## Quick Start
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Apps Included
File Manager, Terminal, Notepad, Calculator, Browser, Paint, Media Player,
Settings, App Store, Recycle Bin.

## Stack
- React 18 + Vite
- Zustand (state)
- localStorage (virtual filesystem & persistence)
- lucide-react (icons)
